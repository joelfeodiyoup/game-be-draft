## domain areas

- games
    - game catalog and metadata
    - public data that everyone can see
- sessions
    - active gameplay sessions (real-time tracking)
    - who is playing what game right now
    - session start/end times
    - real-time event streaming
    - can be queried for 'currently playing' dashboards
- saves
    - saved game state
    - requires authentication to create/update/delete
    - users can only access their own saves
- users - user profiles and authentication
- analytics
    - historical playtime aggregations
    - high scores, completion tracking
    - some data public, some requires authentication?
